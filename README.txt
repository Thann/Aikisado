Aikisado is free software realeased under the GPLv3
that you can it download from http://sourceforge.net/projects/aikisado/

Windows install:-----------------------------------
After installing everything in the installers bundle,
Place Aikisado.py and the GUI folder somewhere safe.
Right-click and select "send to desktop" to create a shortcut.
To run, double click on Aikisado.pyw or the shortcut.

Linux install:-------------------------------------
Make sure pygtk is installed, Most modern distros will already have this
Place the Aikisado Folder somewhere safe and double click Aikisado.pyw
